# GEMCITABINE-OXALIPLATIN - NCI

# GEMCITABINE-OXALIPLATIN

Placeholder slot

(jem-SY-tuh-been-ok-SA-lih-pla-tin)

This chemotherapy combination contains the drugs gemcitabine hydrochloride and oxaliplatin. Combinations usually work better than single drugs because
different drugs kill
cancer
cells in different ways.

Drugs in the GEMCITABINE-OXALIPLATIN combination:

|  |  |
| --- | --- |
|  | [Gemcitabine Hydrochloride](/about-cancer/treatment/drugs/gemcitabinehydrochloride) |
|  | [Oxaliplatin](/about-cancer/treatment/drugs/oxaliplatin) |

Each of the drugs in this combination is approved by the
[Food and Drug
Administration (FDA)](/Common/PopUps/popDefinition.aspx?id=454785&version=Patient&language=English)
to treat cancer or conditions related to cancer.

## Use in Cancer

GEMCITABINE-OXALIPLATIN
is used to treat:

- **[Pancreatic cancer](/Common/PopUps/popDefinition.aspx?id=44521&version=Patient&language=English).**

This combination may also be used with other drugs or
treatments or to treat other types of cancer.
