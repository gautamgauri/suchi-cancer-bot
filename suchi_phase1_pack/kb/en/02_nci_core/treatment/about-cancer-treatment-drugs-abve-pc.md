# ABVE-PC - NCI

# ABVE-PC

This chemotherapy combination contains the drugs doxorubicin hydrochloride (Adriamycin), bleomycin, vincristine sulfate, etoposide phosphate, prednisone, and cyclophosphamide. Combinations usually work better than single drugs because
different drugs kill
cancer
cells in different ways.

Drugs in the ABVE-PC combination:

|  |  |
| --- | --- |
| **A** | = [Doxorubicin Hydrochloride](/about-cancer/treatment/drugs/doxorubicinhydrochloride) (Adriamycin) |
| **B** | = [Bleomycin](/about-cancer/treatment/drugs/bleomycin) |
| **V** | = [Vincristine Sulfate](/about-cancer/treatment/drugs/vincristinesulfate) |
| **E** | = [Etoposide Phosphate](/about-cancer/treatment/drugs/etoposidephosphate) |
| **P** | = [Prednisone](/about-cancer/treatment/drugs/prednisone) |
| **C** | = [Cyclophosphamide](/about-cancer/treatment/drugs/cyclophosphamide) |

Each of the
drugs
in this
combination
is approved by the
[Food and Drug
Administration (FDA)](/Common/PopUps/popDefinition.aspx?id=454785&version=Patient&language=English)
to treat cancer or conditions related to
cancer.

## Use in Cancer

ABVE-PC is used to treat:

- **[Hodgkin lymphoma](/Common/PopUps/popDefinition.aspx?id=270800&version=Patient&language=English)** in children.

This combination may also be used with other drugs or treatments or to treat other types of cancer.
